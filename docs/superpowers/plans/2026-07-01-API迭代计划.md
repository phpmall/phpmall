# API 接口迭代计划实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 8 周内补齐 PHPMall 非 Admin API 通道（User / Shop / Seller / Portal）的 P0+P1 接口，打通 C 端交易闭环与商家后台核心能力，并完成与前端 package 的联调。

**Architecture:** 按业务域拆分迭代（基础设施 → 商品域 → 交易域 → 补齐域），API 层 `app/Api/{Channel}/Controllers` 调用 `app/Modules/{Domain}/Services`；认证统一为 JWT Bearer（RS256）；每个业务域完成后立即与对应前端联调。

**Tech Stack:** Laravel 13 + PHP 8.4 + MySQL 8.4 + Redis + JWT + OpenAPI (Scramble)

**重要约束：**
1. 数据库迁移文件统一放在 Laravel 默认目录 `database/migrations/`，按模块将同模块所有表写入同一个迁移文件。
2. 当前仅 `Auth`、`User` 模块已生成 Service/Repository；其余模块需 Sprint 0 先完成数据表设计 → 运行 `php artisan gen:model/service/dao` 生成代码 → 再进入 API 开发。
3. 整体顺序：**模块数据表设计 → gen 工具生成 Model/Service/Repository → API 接口实现**。

```php
// app/Api/User/Controllers/AuthController.php
public function login(Request $request): JsonResponse
{
    $credentials = $request->only('phone', 'password');
    if (!$token = auth('user')->attempt($credentials)) {
        return $this->error('登录失败', 2001);
    }
    return $this->respondWithToken($token);
}

private function respondWithToken(string $token): JsonResponse
{
    return $this->success([
        'access_token' => $token,
        'token_type' => 'Bearer',
        'expires_in' => auth('user')->factory()->getTTL() * 60,
        'refresh_token' => '', // 见 Task 4
    ]);
}
```

```php
// app/Api/Common/Controllers/AuthController.php
<?php
declare(strict_types=1);

namespace App\Api\Common\Controllers;

use App\Api\Common\Requests\Auth\RefreshRequest;
use Illuminate\Http\JsonResponse;
use Illuminate\Support\Facades\Redis;

class AuthController extends BaseController
{
    public function refresh(RefreshRequest $request): JsonResponse
    {
        $token = auth()->refresh();
        $jti = auth()->payload()->get('jti');

        // 旧 jti 加入黑名单 24h
        $oldJti = auth()->getToken()->getClaim('jti');
        Redis::setex('jwt:blacklist:'.$oldJti, 86400, '1');

        return $this->success([
            'access_token' => $token,
            'token_type' => 'Bearer',
            'expires_in' => auth()->factory()->getTTL() * 60,
        ]);
    }
}
```

```php
public function ossPolicy(OssPolicyRequest $request): JsonResponse
{
    $type = $request->input('type', 'product');
    $dir = $type.'/'.date('Y/m');

    // 根据配置生成 OSS 临时凭证
    $policy = $this->generateOssPolicy($dir);

    return $this->success($policy);
}

private function generateOssPolicy(string $dir): array
{
    $id = config('filesystems.disks.oss.access_key_id');
    $key = config('filesystems.disks.oss.access_key_secret');
    $host = config('filesystems.disks.oss.cdn_domain');
    $bucket = config('filesystems.disks.oss.bucket');

    $expiration = now()->addMinutes(5)->format('Y-m-d\TH:i:s.000\Z');
    $conditions = [
        ['bucket' => $bucket],
        ['content-length-range', 0, 10485760],
        ['starts-with', '$key', $dir.'/'],
    ];

    $policyJson = base64_encode(json_encode(['expiration' => $expiration, 'conditions' => $conditions]));
    $signature = base64_encode(hash_hmac('sha1', $policyJson, $key, true));

    return [
        'access_url' => $host,
        'policy' => $policyJson,
        'signature' => $signature,
        'access_key_id' => $id,
        'dir' => $dir.'/', // @phpstan-ignore-line
        'expires_in' => 300,
    ];
}
```
