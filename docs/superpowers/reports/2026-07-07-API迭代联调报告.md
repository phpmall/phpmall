## 五、延期到 V1.1 / Phase 2+ 的接口

- **Supplier 通道**：入驻、供货商品、采购订单、库存同步、结算对账
- **Finance 域**：钱包余额/流水、充值、提现申请/审核、商家结算
- **Marketing 扩展**：满减活动、秒杀、积分商城
- **Distribution 域**：分销员申请/审核、佣金结算、团队管理
- **Compliance 域**：发票、KYC、电子合同、隐私/同意管理、投诉
- **Logistics 域**：物流公司列表、物流轨迹查询、电子面单
